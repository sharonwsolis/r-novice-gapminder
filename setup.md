This lesson assumes you have the R, RStudio software installed on your computer.

R can be downloaded [here](https://cran.r-project.org/mirrors.html).

RStudio is an environment for developing using R.
It can be downloaded [here](https://www.rstudio.com/products/rstudio/download/).
You will need the Desktop version for your computer.

You also need to download some files to follow this lesson:

1. Make a new folder in your Desktop called `r-novice-gapminder`.
2. Download [r-novice-gapminder-data.zip]({{ page.root }}/data/r-novice-gapminder-data.zip) and move the file to this folder.
3. If it's not unzipped yet, double-click on it to unzip it. You should end up with a new folder called `data`.
4. You can access this folder from the Unix shell with:

~~~
$ cd
$ cd Desktop/r-novice-inflammation/data
~~~
